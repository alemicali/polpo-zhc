# Insurtech Solutions

**MGA Insurance — Ramo Cauzioni e Fideiussioni (Surety)**

Insurtech Solutions è una Managing General Agency (MGA) specializzata nel mercato assicurativo italiano delle cauzioni e fideiussioni. L'azienda gestisce l'intero ciclo di vita delle polizze fideiussorie: dall'analisi del rischio e sottoscrizione, fino alla gestione documentale, compliance normativa e relazioni con la rete distributiva.

🌐 [insurtechsolutions.it](https://insurtechsolutions.it)

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, TypeScript, Python
- **Database:** PostgreSQL
- **Infra:** Docker, GitHub Actions

## Team

Il team è organizzato in 4 aree: Underwriting & Risk, Operations & Finance, Commerciale & Marketing, Sviluppo & Tech.
