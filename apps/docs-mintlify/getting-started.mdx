title: "Quickstart"
description: "From zero to running agents in under 2 minutes."
---

<Note>
  No API key needed. The default model is free.
</Note>

<Steps>

1. **Install**

   <Tabs>
     <Tab title="npm">
       ```bash
       npm install -g openpolpo
       ```
     </Tab>
     <Tab title="pnpm">
       ```bash
       pnpm add -g openpolpo
       ```
     </Tab>
     <Tab title="Docker">
       ```bash
       docker pull ghcr.io/openpolpo/openpolpo:latest
       ```
       Then skip to step 4 — run with `docker run -p 3000:3000 -v $(pwd):/workspace ghcr.io/openpolpo/openpolpo:latest`
     </Tab>
   </Tabs>

2. **Init a project**

   ```bash
   mkdir my-project && cd my-project
   polpo init
   ```

   This creates `.polpo/polpo.json` with a default team.

3. **Launch the TUI**

   ```bash
   polpo
   ```

   You're in. Tell Polpo what you need, hit Enter, and watch it work.

4. **Or go headless**

   ```bash
   polpo serve --port 3000
   ```

   Full REST API at `http://localhost:3000/api/v1/` — create tasks, execute plans, stream events via SSE.

</Steps>

## Try a plan

Create `.polpo/plans/hello.json`:

```json
{
  "name": "hello-world",
  "group": "hello",
  "tasks": [
    {
      "title": "Create a greeting",
      "agent": "default",
      "description": "Create a file called hello.txt that says 'Hello from Polpo!' in a creative way."
    }
  ]
}
```

Run it:

```bash
polpo run
```

Polpo picks the right agent, runs the task, and writes `hello.txt` to your project directory.

## Add an API key (optional)

Want to use Claude, GPT-4, or Gemini instead of the free default?

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

Then set the model in `.polpo/polpo.json`:

```json
{
  "team": {
    "agents": [
      {
        "name": "default",
        "model": "anthropic:claude-sonnet-4-5-20250929",
        "role": "general-purpose developer"
      }
    ]
  }
}
```
