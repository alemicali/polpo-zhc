Polpo supports **22+ LLM providers** out of the box. Pick a provider, set the API key, then use `provider:model` in your config.

## Quick Start

1. Set the API key **or** login via OAuth:

<CodeGroup>
```bash API Key
export ANTHROPIC_API_KEY=sk-ant-...
```

```bash OAuth (Claude Max)
polpo auth login anthropic
```
</CodeGroup>

2. Reference the model in your agent config:

```json
{
  "agents": [
    { "name": "coder", "model": "anthropic:claude-sonnet-4-5-20250929" }
  ]
}
```

For full details on model format, resolution order, and fallback chains, see [Overview](/models/overview). For OAuth setup and auth profiles, see [Authentication](/models/authentication).

## Cloud Providers

- [Anthropic (Claude)](/providers/anthropic) — industry-leading coding and reasoning
- [OpenAI (GPT, o-series)](/providers/openai) — multimodal flagship and reasoning models
- [Google Gemini](/providers/google) — massive context windows (1M+ tokens)
- [Google Vertex AI](/providers/google-vertex) — enterprise Gemini with VPC-SC, CMEK
- [Amazon Bedrock](/providers/amazon-bedrock) — Claude, Llama, Mistral through AWS
- [Azure OpenAI](/providers/azure-openai) — OpenAI models with Azure compliance
- [Groq](/providers/groq) — ultra-fast inference on custom LPU hardware
- [Cerebras](/providers/cerebras) — wafer-scale inference, extremely fast
- [Mistral](/providers/mistral) — code-specialized Codestral and Devstral
- [xAI (Grok)](/providers/xai) — Grok models, competitively priced

## Gateways & Aggregators

- [OpenRouter](/providers/openrouter) — meta-gateway to hundreds of models (DeepSeek, Cohere, etc.)
- [Vercel AI Gateway](/providers/vercel-ai-gateway) — unified routing with observability
- [LiteLLM](/providers/litellm) — proxy that unifies 100+ providers behind one endpoint

## Subscription & OAuth

- [GitHub Copilot](/providers/github-copilot) — use your Copilot subscription
- [OpenCode](/providers/opencode) — free default provider, no API key needed

## Regional & Specialty

- [Hugging Face](/providers/huggingface) — open models via HF Inference API
- [MiniMax](/providers/minimax) — global and China endpoints
- [Kimi / Moonshot](/providers/kimi) — code-specialized K2 model
- [ZAI / GLM](/providers/zai) — Zhipu AI, strong bilingual Chinese/English

## Local & Self-Hosted

Run models on your own hardware — zero API cost, full data privacy.

- [Ollama](/providers/ollama) — easiest way to run local models
- [vLLM](/providers/vllm) — high-throughput production inference
- [LM Studio](/providers/lmstudio) — desktop GUI with built-in server
- [TGI](/providers/tgi) — Hugging Face's production inference server

All local providers use the `openai-completions` API mode and support auto-discovery via `polpo models scan`.

For custom provider setup details, API compatibility modes, and model definitions, see [Custom Providers](/models/custom-providers).
